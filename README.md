## Installation

```sh

# Clone the repo
$ git clone git remote add origin https://github.com/a25us8GingerMediaGroup_Assignment.git

# Install
$ npm install

# Setting Up ENV
> setup .env file for database



# Start
$ npm start

# Access your app
$ http://localhost:${PORT}
```
